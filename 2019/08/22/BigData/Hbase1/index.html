Hbase的安装

1.下载Hbase

http://mirror.bit.edu.cn/apache/hbase/2.1.5/

2.解压至本地路径

[root@master ~]# tar -xzvf hbase-2.1.5-bin.tar.gz -C /usr/local/

3.修改Hbase配置文件

切换路径

[root@master ~]# cd /usr/local/hbase-2.1.5/conf

3.1 修改hbase-env.sh脚本，添加以下信息

[root@master conf]# vi hbase-env.sh

3.2 修改hbase-site.xml

[root@master conf]# vi hbase-site.xml

3.3 修改regionservers

[root@master conf]# vi regionservers

4.启动Hbase集群

检查zookeeper状态

[root@master ～]# zkServer.sh status

检查namenode,datanode状态

[root@master ～]# jps
http://master:50070/ 登陆网页查看

检查完后，启动Hbase

[root@master ～]# start-hbase.sh

查看是否成功启动

http://master:16010/ 登陆网页查看Hbase状态
[root@master ～]# hbase shell 启动hbase命令行客户端

注释 Hmaster进程启动后消失

启动hbase报java.lang.ClassNotFoundException: org.apache.htrace.SamplerBuilder
解决方法
cp $HBASE_HOME/lib/client-facing-thirdparty/htrace-core-3.1.0-incubating.jar $HBASE_HOME/lib/
