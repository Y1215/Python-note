Zookeeper安装

发表于 2019-08-22 | 分类于 BigData ， Zookeeper

Zookeeper的安装

1.下载Zookeeper安装包

http://archive.apache.org/dist/zookeeper/zookeeper-3.4.2/

2.解压至本地

[root@master ~]# tar -xzvf zookeeper-3.4.2.tar.gz -C /usr/local/

3.配置环境变量

vi /etc/profile
在文件末尾添加

使配置生效

[root@master ~]# source /etc/profile

4.配置Zookeeper文件

切换路径

[root@master ~]# cd /usr/local/zookeeper-3.4.2/conf/

4.1 复制zoo_sample.cfg模板配置文件为正式的配置文件zoo.cfg

[root@master ~]# cp zoo_sample.cfg zoo.cfg

4.2 创建myid文件

切换路径

[root@master conf]# cd /usr/local/zookeeper-3.4.2/data
创建myid文件,并根据zoo.cfg中的配置内容，将本机与server.x对应的x写入myid文件中
[root@master data]# vi myid
例如本机为master，zoo.cfg配置文件中为server.1，则在myid文件中写入1

4.3 复制Zookeeper

[root@master data]# scp -r /usr/local/zookeeper-3.4.2/ slave1:/usr/local/
[root@master data]# scp -r /usr/local/zookeeper-3.4.2/ slave2:/usr/local/
[root@master data]# scp -r /usr/local/zookeeper-3.4.2/ slave3:/usr/local/

4.4 修改slave1，2，3的myid

将对应的id写入myid中

4.5 复制环境变量配置文件

[root@master data]# scp /etc/profile slave1:/etc
[root@master data]# scp /etc/profile slave2:/etc
[root@master data]# scp /etc/profile slave3:/etc

切换至slave，使配置文件生效

[root@slave1 ~]# source /etc/profile

5.启动Zookeeper集群

需要每台机器都执行

[root@master ～]# zkServer.sh start
