Sqoop的使用

1.列出所有的数据库

[root@master bin]# sqoop list-databases --connect jdbc:mysql://localhost:3306/ --username root --password 12345678

2.测试数据库

3.从mysql导入到HDFS指定目录

使用参数 --target-dir来指定导出目的地，
使用参数 --delete-target-dir来判断导出目录是否存在，如果存在就删掉

[root@master ～]# sqoop import --connect jdbc:mysql://192.168.0.130:3306/userdb --username root --password 12345678 --delete-target-dir --table emp --target-dir /sqoop/emp --m 1

查看导出的数据

默认数据之间分隔符为”,”,自定义分隔符

[root@master ～]# sqoop import --connect jdbc:mysql://192.168.0.130:3306/userdb --username root --password 12345678 --delete-target-dir --table emp --target-dir /sqoop/emp2 --m 1 --fields-terminated-by '\t'

4.mysql导入到HIVE

4.1 导入所需jar包

[root@master ~]# cp /usr/local/hive-2.3.5/lib/hive-exec-2.3.5.jar /usr/local/sqoop-1.4.6/lib/

4.2 在Hive中创建数据库与表

4.3 导入数据

[root@master ~]# sqoop import --connect jdbc:mysql://192.168.0.130:3306/userdb --username root --password 12345678 --table emp --fields-terminated-by '\001' --hive-import --hive-table sqooptohive.emp_hive --hive-overwrite --delete-target-dir --m 1

4.4 hive表查看

hive> select * from emp_hive;

4.5 导入关系表到hive并自动创建hive表

[root@master ~]# sqoop import --connect jdbc:mysql://192.168.0.130:3306/userdb --username root --password 12345678 --table emp_conn --hive-import -m 1 --hive-database sqooptohive;

5.导入hdfs

5.1 导入表数据子集

使用”where”子句的一个子集，它执行在各自的数据库服务器相应的SQL查询，并将结果存储在HDFS的目标目录。
where子句的语法——--where <condition>

  • 按照条件进行查找，通过--where参数来查找表emp_add当中city字段的值为sec-bad的所有数据导入到hdfs上面去

    [root@master ~]# sqoop import --connect jdbc:mysql://192.168.0.130:3306/userdb --username root --password 12345678 --table emp_add --target-dir /sqoop/emp_add -m 1 --delete-target-dir --where "city = 'sec-bad'"

5.2 sql语句查找导入hdfs

通过-–query参数来指定我们的sql语句，通过sql语句来过滤数据进行导入

[root@master ~]# sqoop import --connect jdbc:mysql://192.168.0.130:3306/userdb --username root --password 12345678 --delete-target-dir -m 1 --query 'select phno from emp_conn where 1=1 and $CONDITIONS' --target-dir /sqoop/emp_conn

  • 注意

    • 使用sql语句来进行查找是不能加参数–table
    • 必须要添加where条件，
    • where条件后面必须带一个$CONDITIONS 这个字符串，
    • sql语句必须用单引号，不能用双引号

  • 查看hdfs上的内容

5.3 增量导入

Sqoop导入命令增量选项的语法

5.3.1 使用上面的选项来实现

  • 导入emp表当中id大于1202的所有数据

    • 注意：增量导入的时候，一定不能加参数–delete-target-dir否则会报错

      [root@master ~]# sqoop import --connect jdbc:mysql://192.168.0.130:3306/userdb --username root --password 12345678 --table emp --incremental append --check-column id --last-value 1202 -m 1 --target-dir /sqoop/increment

  • 查看数据内容

5.3.2 通过--where条件来实现——选取的数据会更加精准

[root@master ~]# sqoop import --connect jdbc:mysql://192.168.0.130:3306/userdb --username root --password 12345678 --table emp --incremental append --where "create_time > '2019-06-17 00:00:00' and is_delete='1' and create_time < '2019-09-17 23:59:59'" --target-dir /sqoop/incement2 --check-column id --m 1

6.将数据从HDFS把文件导出到MYSQL

  • 注意
    • 导出前，目标表必须存在于目标数据库中。
    • 默认操作是从将文件中的数据使用INSERT语句插入到表中
    • 更新模式下，是生成UPDATE语句更新表数据

      6.1 查看hdfs目录/sqoop/emp下的数据内容

6.2 创建mysql表

mysql>

6.3 执行导出命令

[root@master ~]# sqoop export --connect jdbc:mysql://192.168.0.130:3306/userdb --username root --password 12345678 --table emp_out --export-dir /sqoop/emp --input-fields-terminated-by ","

6.4 验证mysql表数据

mysql> select * from emp_out
