Hadoop集群搭建

1.切换root用户

  • CentOS7修改root用户密码
    • [root@localhost ~]# sudo passwd
    • Password：当前用户密码
    • New password：root密码
    • Retype new password: 重复root密码

2.修改主机名字(CentOS7)

  • 查看当前的 hostnmae
    • [root@localhost ~]# hostname
  • 永久修改hostname(立即生效）
    • [root@localhost ~]# hostnamectl set-hostname master
  • 检查修改
    • [root@localhost ~]# hostname

3.修改网络配置

vi /etc/sysconfig/network-scripts/ifcfg-eth0

删掉这两句
DEFROUTE=yes
PEERDNS=yes

修改
BOOTPROTO=static

添加
PROXY_METHOD=none
BROWSER_ONLY=no
IPADDR=192.168.0.130 //本机IP地址
GATEWAY=192.168.0.1 //子网掩码
PREFIX=255.255.255.0 //网关
DNS1=114.114.114.114 //DNS

4.配置hosts

vi /etc/hosts 添加主机与服务器节点信息

将配置好的hosts文件发送至其他服务器

scp /etc/hosts root@slave1:/etc/
scp /etc/hosts root@slave2:/etc/
scp /etc/hosts root@slave3:/etc/

5.关闭防火墙

查看防火墙状态
firewall-cmd --state
关闭防火墙
systemctl stop firewalld.service
禁止防火墙开机启动
systemctl disable firewalld.service

6.时间同步

下载ntp插件
[root@master ~]# yum install ntp
同步中国上海时间，用阿里云服务器
[root@master ~]# ntpdate -u ntp1.aliyun.com
查看是否配置成功
[root@master ~]# date
