Flume安装

1.下载并解压至/usr/local/

http://archive.apache.org/dist/flume/1.7.0/apache-flume-1.7.0-bin.tar.gz

2.配置flume-env.sh

[root@master ~]# cd /usr/local/flume-1.7.0/conf/
[root@master conf]# cp flume-env.sh.template flume-env.sh
[root@master conf]# vi flume-env.sh

添加信息
export JAVA_HOME=/usr/local/jdk1.8.0_221

3.开发配置文件

[root@master conf]# vi /usr/local/flume-1.7.0/conf/netcat-logger.conf

4.启动配置文件

指定采集方案配置文件，在相应的节点上启动flume agent

[root@master ~]# bin/flume-ng agent -c conf -f conf/netcat-logger.conf -n a1 -Dflume.root.logger=INFO,console

5.测试

在slave1安装telnet

[root@slave1 ~]# yum -y install telnet
