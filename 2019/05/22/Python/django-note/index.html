ubuntu中创建django
#### ubuntu中创建django项目 1.安装pip —— sudo apt install python-pip 安装pip3 —— sudo apt install python-pip3

2.创建虚拟环境

  • 安装virtualenv —— pip install virtualenv
  • 创建虚拟环境 —— virtualenv -p 路径 虚拟环境的名字 或 virtualenv 虚拟环境的命令
  • 激活虚拟环境 —— source py3env/bin/activate
  • 安装django —— pip install django==版本号
  • 退出虚拟环境，输入命令 —— deactivate

3.新建一个django项目

  • django-admin.py startproject 项目名称
  • django-admin.py startapp app名称
  • 在pycharm 打开该项目
  • 添加虚拟环境
    (1)找到setting
    (2)在setting找到Project
    (3)project interpreter
    (4)添加add
    (5)选择Exixting enviroment 找到路径
    (6)apply –> ok
  • 启动项目 —— 找到terminal 输入 python manage.py runserver 指定ip:指定端口

4.app

  • 在app文件夹下创建urls.py
    • urlpatterns = [url(r’^路径名称/‘, 名称),]
  • settings.py
    • INSTALLED_APPS 添加app

5.project

  • 项目文件夹下的urls.py中添加url(r’^app名称/‘,include(app名称.urls)),

6.templates

  • 创建templates文件夹，在文件夹下创建html文件
  • 在settings.py下 TEMPLATES
    • ‘DIRS’: [os.path.join(BASE_DIR, ‘templates’)],
